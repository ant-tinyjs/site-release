PointAttachment

.spine. PointAttachment

An attachment which is a single point and a rotation. This can be used to spawn projectiles, particles, etc. A bone can be used in similar ways, but a PointAttachment is slightly less expensive to compute and can be hidden, shown, and placed in a skin. See [Point Attachments](http://esotericsoftware.com/spine-point-attachments) in the Spine User Guide.
